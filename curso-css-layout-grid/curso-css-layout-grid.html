Curso CSS: Construindo layout com GRID


Data de conclusão: __/__/____

Projeto do Curso


Aula 1: Primeiros passos
Aplicar o display grid corretamente para organizar grupos de elementos;
Construir áreas em um layout com grid-template-areas e grid-area;
Definir largura de colunas com grid-template-columns;
Aplicar responsividade unindo grid e media queries.

Aula 2: Avançando no GRID
Criar e definir a altura de linhas com grid-template-rows;
Utilizar as propriedades gap, column-gap e row-gap para aplicar espaçamento em grupos de itens;
Aplicar propriedades como align-items e justify-self para alinhar itens.

Aula 3: GRID Layout
Organizar elementos de forma responsiva unindo grid, auto-fit e min-max;
Organizar itens determinando a altura de linhas e largura de colunas em conjunto;
Determinar a posição de itens através da sua posição inicial e final em um grid com grid-area.

Aula 4: Criando GRID com barra de rolagem
Organizar itens com Grid em uma mesma linha com grid-auto-flow;
Aplicar barra de rolagem permitindo visualização de todos os itens em um container;
Definir a posição inicial e final de uma área de itens com grid-column.
